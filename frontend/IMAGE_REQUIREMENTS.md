## Style Guidelines

### Color Scheme
- **Primary**: Laboratory blues and whites
- **Accents**: Safety yellows and reds
- **Liquids**: Transparent with subtle coloring
- **Metals**: Realistic metallic grays and silvers

### Art Style
- Semi-realistic scientific illustrations
- Clean, professional appearance
- Consistent lighting and shadows
- Medical/laboratory aesthetic

### Technical Requirements
- SVG format preferred for scalability
- PNG fallback for complex illustrations
- Transparent backgrounds for overlay
- Consistent sizing and proportions
- Web-optimized file sizes

## Priority Order

1. **High Priority**: Main laboratory background, basic apparatus (beakers, thermometers, hotplate)
2. **Medium Priority**: Ingredient containers, control panels, protocol icons
3. **Low Priority**: Background elements, safety equipment, detailed views
4. **Optional**: Animation frames, molecular structures, microscopic views

## Notes for Illustrator

- Maintain scientific accuracy in apparatus representations
- Ensure interactive elements are clearly distinguishable
- Use consistent lighting and shadow directions
- Consider color-blind accessibility
- Keep file sizes optimized for web performance
- Provide both light and dark theme variants if possible
